hexo-admin

就目前而言，发布hexo博客的方式有如下几种：

  • 直接在服务器上编写，然后生成静态文件后部署
  • 利用github+hook来管理，教程点击这里
  • 其他（hexo或者other plugins）

这里推荐使用 hexo-admin，而且编辑完之后可以马上看到效果呢。 需要说明的是，hexo-admin 管理是本地用的，就是你需要在本地编辑完之后再上传到 github，而不能直接在线编辑保存，因为 github pages 只支持静态页面的。

  2. 启动 启动站点，你就可以在localhost://4000/admin/下面看到管理界面了。
