简单介绍一下SVG

什么是SVG

  • 使用XML定义的可伸缩的矢量图形

svg的优点

  • 图像可伸缩且质量不会因为伸缩受到影响
  • 可被更多的工具读取和修改
  • 相对于图片，尺寸更小，且可压缩性更小

svg常用属性

注：svg的属性都可以写在css中

  • fill
  • stroke
  • stroke-width
  • stroke-dasharray
  • stroke-dashoffset

svg形状

  • 直线 <line x1="" y1="" x2="" y2="" />
  • 折线 <polyline point="" />
  • 矩形 <rect x="" y="" width="" height="" />
  • 圆形 <circle cx="" cy="" r="" />
  • 多边形 <polyon point="" />

svg路径

svg的路径用path标签来定义。 常用命令：

  • M = moveto 移动到某个坐标
  • L = lineto 画一条直线
  • C = curveto 三次贝塞尔曲线
  • Q = quadratic Belzier curve 二次贝塞尔曲线
  • Z = closepath 闭合路径

path标签在网页中的应用非常广泛，后面会说到。

甩锅给UI系列二之SVG描边动画

stroke家族终于要隆重登场啦👏👏👏

没错，实现描边动画，只需要使用到stroke、stroke-width、stroke-dasharray、stroke-dashoffset等属性

下面，我会用一个简单的🌰来一步步讲解描边动画的实现原理

先来看一下我们最终要实现的动画效果

  1. 先用AI做出基本的描边图形 我们这里先画出一个键盘的图形，当然，画图这种事情，还是要交给设计小姐姐小哥哥了。

  2. 导出svg，得到svg代码。观察动画，我们发现动画是先从键盘开始描边，然后是字母键，回车键，空格键，最后是线，所以为了方便处理，设计会将键盘、字母键、回车键、空格键、线分别画在不同的图层中，这样导出来的svg代码便是以<g></g>标签分组的，便于我们整体操作。所以，在设计画图之前，要注意与设计之间的沟通。

  3. 给border、keyboard、enter、space、line设置css,将每一块的stroke-dasharray设置为图形周长，stroke-dashoffset设置为周长，设置完毕后，所有图层在页面上都是不可见状态。

  4. 设置关键帧动画：每个图层最终都会以完整的形状出现在页面上，所以，只需要在最后关键帧，将stroke-dasharray设置为0即可。

  5. 使用动画。因为每个图层是上一图层描边完成才开始下一个图层的描边，所以要给后面的图层动画设置延时。

这样，我们就完成了一个图形描边动画了。

其实这是跟图形描边动画一样的原理，叫做文字描边动画.

文字描边动画跟图形区别在于，文字的路径长度相对于规则的图形来说更难计算，且需要控制每个字母的变化，相对繁琐。但是原理相同，都是通过stroke-dasharray以及stroke-dashoffset加上css3动画完成效果。
